你好！我是你的物理考试教练。很高兴看到你开始攻克这些计算题。

既然你的微积分刚起步，不要被“积分”吓倒。在物理考试中，积分通常只是一个固定的**“套路”**。

这三道题（4、5、6）分别代表了静电场计算中三种最典型的“套路”：

  • 第4题：直棒模型（一维积分，最基础）。
  • 第5题：圆环/圆弧模型（利用对称性简化积分）。
  • 第6题：圆盘与无限大平板模型（利用公式拼接，考逻辑而非积分）。

由于内容较多，为了保证你完全听懂，我们一次性把这三个“必考模型”全部讲透。

第一部分：核心概念与“傻瓜式”公式

在做题前，我们需要先准备好工具。对于这几道题，你只需要掌握以下核心概念：

1.1 把电荷切成“微元” (dq)

因为库仑定律只对“点电荷”有效，而棒子、圆环是一大坨电荷。我们的策略是：切！ 把物体切成无数个无限小的碎片，每一个碎片就是一个点电荷 dq，算出它的电场，然后加起来（积分）。

  • 线密度 λ (Lambda)：电荷分布在一条线上。 dq=λdl (电荷量 = 密度 × 长度微元)
  • 面密度 σ (Sigma)：电荷分布在一个面上。 dq=σdS (电荷量 = 密度 × 面积微元)

1.2 点电荷的电场与库仑力公式

  • 库仑力 (力)： F=kr2q1​q2​​
  • 电场强度 (场)： E=kr2q​ 其中 k 是静电力常量。

1.3 两个直接背诵的“二级结论”公式 (针对第6题)

考试时直接推导太慢，这两个公式建议直接当单词背下来：

  1. 均匀带电圆盘轴线上的电场公式： E圆盘​=2ε0​σ​(1−cosθ) 其中 θ 是圆盘边缘对场点的张角（半顶角），cosθ=x2+R2​x​。
  2. 无限大均匀带电平面的电场公式： 当圆盘半径 R→∞ 时，θ→90∘，cosθ→0，公式变为： E无限大​=2ε0​σ​

第二部分：详细解题步骤

第4题：带电细杆对点电荷的作用力

2.1 题目分析

  • 已知：细杆长 l，电荷线密度 λ，点电荷 q0​，距离 d。
  • 思路：把杆切成无数小段 dx，每一段看作点电荷，算出它对 q0​ 的力，然后积分。

2.2 解题步骤

  1. 建立坐标系： 以点电荷 q0​ 所在位置为原点 O，向右为 x 轴正方向。 细杆的左端坐标为 d，右端坐标为 d+l。

  2. 取微元： 在杆上坐标为 x 处，取一段极短的长度 dx。 这一小段上的电荷量为： dq=λdx

  3. 列出微元力 dF： 这一小段电荷 dq 与点电荷 q0​ 的距离就是 x。根据库仑定律： dF=kx2q0​⋅dq​=kx2q0​λdx​

  4. 积分求总力： 把所有小段的力加起来，范围从杆头 (x=d) 到杆尾 (x=d+l)。 F=∫dd+l​kq0​λx21​dx

    提常数： F=kq0​λ∫dd+l​x−2dx

    计算积分 (基本公式 ∫x−2dx=−x−1): F=kq0​λ[−x1​]dd+l​

    代入上下限： F=kq0​λ((−d+l1​)−(−d1​)) F=kq0​λ(d1​−d+l1​)

    这就是最终答案。 (如果需要通分也可以写成 F=d(d+l)kq0​λl​，因为 Q=λl，也可以写成 F=d(d+l)kq0​Q​)。

第5题：均匀带电半圆环的圆心场强

2.1 题目分析

  • 已知：半圆环半径 R，总电荷 Q (正电荷)，求圆心 O 处的 E。
  • 思路：利用对称性。左边的电荷想把 O 点往右推，右边的想往左推，水平方向抵消了！只剩下垂直方向的分量。

2.2 解题步骤

  1. 计算线密度： 半圆的长度是 πR。 λ=πRQ​

  2. 取微元： 在圆环上取一小段弧长 dl，对应的角度微元是 dθ。 dl=Rdθ 电荷量 dq=λdl=λRdθ。

  3. 写出微元场强 dE： 这一小点在圆心产生的场强大小： dE=kR2dq​=kR2λRdθ​=Rkλ​dθ

  4. 分析分量 (关键一步)： 建立坐标系，让半圆环对称轴沿 y 轴。 由对称性可知，x 轴方向场强为 0。 我们只算 y 轴分量：dEy​=dEsinθ (假设 θ 是与 x 轴的夹角)。

  5. 积分： θ 从 0 积到 π (0 到 180度)。 E=∫0π​dEsinθ=∫0π​Rkλ​sinθdθ

    提常数： E=Rkλ​∫0π​sinθdθ

    计算积分 (∫sinθdθ=−cosθ)： E=Rkλ​[−cosθ]0π​=Rkλ​(−cosπ−(−cos0))

    因为 cosπ=−1,cos0=1： E=Rkλ​(1−(−1))=R2kλ​

    如果题目给的是 Q 而不是 λ，把第一步的 λ=πRQ​ 代入： E=R2k​⋅πRQ​=πR22kQ​

第6题：无限大平面挖孔问题

2.1 题目分析

  • 题目解读：这道题有点像绕口令。“该点场强大小的一半，是由平面上一个半径为 R 的圆面积范围内的电荷产生的”。
  • 翻译成人话：在距离无限大平面 a 的地方，整个无限大平面产生的电场是 E总​。而在平面上画一个半径为 R 的圆盘，这个圆盘单独产生的电场 E圆盘​，刚好等于 E总​ 的一半。求 R 是多少。
  • 思路：列方程，E圆盘​=21​E总​。

2.2 解题步骤

  1. 写出无限大平面的场强公式： E总​=2ε0​σ​

  2. 写出圆盘的场强公式： E圆盘​=2ε0​σ​(1−a2+R2​a​) (注：这里 a2+R2​a​ 就是 cosθ，其中 θ 是圆锥半顶角)

  3. 根据题意列方程： 题目说 E圆盘​=21​E总​，代入公式： 2ε0​σ​(1−a2+R2​a​)=21​⋅2ε0​σ​

  4. 化简求解： 两边同时消去 2ε0​σ​： 1−a2+R2​a​=21​

    移项： a2+R2​a​=21​

    两边平方： a2+R2a2​=41​

    交叉相乘： 4a2=a2+R2 3a2=R2 R=3​a

    答案是 R=3​a，与参考答案一致。

第三部分：考前速记总结 (可直接复制到笔记)

这里为你整理了可以直接背诵和模仿的解题模板。

1. 核心公式表

  • 库仑定律：F=kr2q1​q2​​
  • 电场定义：E=q0​F​=kr2Q​
  • 无限大均匀带电平面场强：E=2ε0​σ​
  • 均匀带电圆盘轴线场强：E=2ε0​σ​(1−cosθ)，其中 cosθ=x2+R2​x​

2. 第4题：直杆与点电荷 (一维积分)

  • 模型：长 l 的带电杆，距离 d 处的点电荷。
  • 关键步骤：
    1. 设变量 x 为微元到点电荷的距离。
    2. 微元电荷 dq=λdx。
    3. 积分区间：[d,d+l]。
  • 计算式： F=∫dd+l​kx2q0​λ​dx=kq0​λ[x−1​]dd+l​
  • 最终结果： F=kq0​λ(d1​−d+l1​)

3. 第5题：半圆环圆心场强 (对称性积分)

  • 模型：半径 R，带电量 Q 或密度 λ 的半圆。
  • 关键步骤：
    1. 对称性分析：平行直径的分量抵消，只算垂直分量。
    2. 垂直分量微元：dEy​=dEsinθ=R2kλRdθ​sinθ。
    3. 积分区间：[0,π]。
  • 最终结果： E=R2kλ​ (若已知总电荷 Q，则 E=πR22kQ​)

4. 第6题：平面与圆盘 (比例关系)

  • 逻辑：题目问“一半场强对应的半径”。
  • 方程： 1−a2+R2​a​=21​
  • 结论： R=3​a
