# 大物22
你好！我是你的物理考试教练。很高兴看到你开始攻克这些计算题。

既然你的微积分刚起步，不要被“积分”吓倒。在物理考试中，积分通常只是一个固定的**“套路”**。

这三道题（4、5、6）分别代表了静电场计算中三种最典型的“套路”：
*   **第4题**：**直棒模型**（一维积分，最基础）。
*   **第5题**：**圆环/圆弧模型**（利用对称性简化积分）。
*   **第6题**：**圆盘与无限大平板模型**（利用公式拼接，考逻辑而非积分）。

由于内容较多，为了保证你完全听懂，我们一次性把这三个“必考模型”全部讲透。

### 第一部分：核心概念与“傻瓜式”公式

在做题前，我们需要先准备好工具。对于这几道题，你只需要掌握以下核心概念：

#### 1.1 把电荷切成“微元” ($dq$)
因为库仑定律只对“点电荷”有效，而棒子、圆环是一大坨电荷。我们的策略是：**切！**
把物体切成无数个无限小的碎片，每一个碎片就是一个点电荷 $dq$，算出它的电场，然后加起来（积分）。

*   **线密度 $\lambda$ (Lambda)**：电荷分布在一条线上。
    $$dq = \lambda dl$$
    (电荷量 = 密度 $\times$ 长度微元)
*   **面密度 $\sigma$ (Sigma)**：电荷分布在一个面上。
    $$dq = \sigma dS$$
    (电荷量 = 密度 $\times$ 面积微元)

#### 1.2 点电荷的电场与库仑力公式
*   **库仑力** (力)：
    $$F = k \frac{q_1 q_2}{r^2}$$
*   **电场强度** (场)：
    $$E = k \frac{q}{r^2}$$
    其中 $k$ 是静电力常量。

#### 1.3 两个直接背诵的“二级结论”公式 (针对第6题)
考试时直接推导太慢，这两个公式建议直接当单词背下来：
1.  **均匀带电圆盘轴线上的电场公式**：
    $$E_{\text{圆盘}} = \frac{\sigma}{2\varepsilon_0} (1 - \cos\theta)$$
    其中 $\theta$ 是圆盘边缘对场点的张角（半顶角），$\cos\theta = \frac{x}{\sqrt{x^2+R^2}}$。
2.  **无限大均匀带电平面的电场公式**：
    当圆盘半径 $R \to \infty$ 时，$\theta \to 90^\circ$，$\cos\theta \to 0$，公式变为：
    $$E_{\text{无限大}} = \frac{\sigma}{2\varepsilon_0}$$

### 第二部分：详细解题步骤

#### 第4题：带电细杆对点电荷的作用力

**2.1 题目分析**
*   **已知**：细杆长 $l$，电荷线密度 $\lambda$，点电荷 $q_0$，距离 $d$。
*   **思路**：把杆切成无数小段 $dx$，每一段看作点电荷，算出它对 $q_0$ 的力，然后积分。

**2.2 解题步骤**

1.  **建立坐标系**：
    以点电荷 $q_0$ 所在位置为原点 $O$，向右为 $x$ 轴正方向。
    细杆的左端坐标为 $d$，右端坐标为 $d+l$。

2.  **取微元**：
    在杆上坐标为 $x$ 处，取一段极短的长度 $dx$。
    这一小段上的电荷量为：
    $$dq = \lambda dx$$

3.  **列出微元力 $dF$**：
    这一小段电荷 $dq$ 与点电荷 $q_0$ 的距离就是 $x$。根据库仑定律：
    $$dF = k \frac{q_0 \cdot dq}{x^2} = k \frac{q_0 \lambda dx}{x^2}$$

4.  **积分求总力**：
    把所有小段的力加起来，范围从杆头 ($x=d$) 到杆尾 ($x=d+l$)。
    $$F = \int_{d}^{d+l} k q_0 \lambda \frac{1}{x^2} dx$$
    
    提常数：
    $$F = k q_0 \lambda \int_{d}^{d+l} x^{-2} dx$$
    
    计算积分 (基本公式 $\int x^{-2} dx = -x^{-1}$):
    $$F = k q_0 \lambda \left[ -\frac{1}{x} \right]_{d}^{d+l}$$
    
    代入上下限：
    $$F = k q_0 \lambda \left( (-\frac{1}{d+l}) - (-\frac{1}{d}) \right)$$
    $$F = k q_0 \lambda \left( \frac{1}{d} - \frac{1}{d+l} \right)$$

    **这就是最终答案。** (如果需要通分也可以写成 $F = \frac{k q_0 \lambda l}{d(d+l)}$，因为 $Q=\lambda l$，也可以写成 $F = \frac{k q_0 Q}{d(d+l)}$)。

#### 第5题：均匀带电半圆环的圆心场强

**2.1 题目分析**
*   **已知**：半圆环半径 $R$，总电荷 $Q$ (正电荷)，求圆心 $O$ 处的 $E$。
*   **思路**：利用**对称性**。左边的电荷想把 $O$ 点往右推，右边的想往左推，水平方向抵消了！只剩下垂直方向的分量。

**2.2 解题步骤**

1.  **计算线密度**：
    半圆的长度是 $\pi R$。
    $$\lambda = \frac{Q}{\pi R}$$

2.  **取微元**：
    在圆环上取一小段弧长 $dl$，对应的角度微元是 $d\theta$。
    $$dl = R d\theta$$
    电荷量 $dq = \lambda dl = \lambda R d\theta$。

3.  **写出微元场强 $dE$**：
    这一小点在圆心产生的场强大小：
    $$dE = k \frac{dq}{R^2} = k \frac{\lambda R d\theta}{R^2} = \frac{k\lambda}{R} d\theta$$

4.  **分析分量 (关键一步)**：
    建立坐标系，让半圆环对称轴沿 $y$ 轴。
    由对称性可知，**$x$ 轴方向场强为 0**。
    我们只算 $y$ 轴分量：$dE_y = dE \sin\theta$ (假设 $\theta$ 是与 $x$ 轴的夹角)。

5.  **积分**：
    $\theta$ 从 $0$ 积到 $\pi$ (0 到 180度)。
    $$E = \int_{0}^{\pi} dE \sin\theta = \int_{0}^{\pi} \frac{k\lambda}{R} \sin\theta d\theta$$
    
    提常数：
    $$E = \frac{k\lambda}{R} \int_{0}^{\pi} \sin\theta d\theta$$
    
    计算积分 ($\int \sin\theta d\theta = -\cos\theta$)：
    $$E = \frac{k\lambda}{R} [-\cos\theta]_{0}^{\pi} = \frac{k\lambda}{R} (-\cos\pi - (-\cos 0))$$
    
    因为 $\cos\pi = -1, \cos 0 = 1$：
    $$E = \frac{k\lambda}{R} (1 - (-1)) = \frac{2k\lambda}{R}$$

    如果题目给的是 $Q$ 而不是 $\lambda$，把第一步的 $\lambda = \frac{Q}{\pi R}$ 代入：
    $$E = \frac{2k}{R} \cdot \frac{Q}{\pi R} = \frac{2kQ}{\pi R^2}$$

#### 第6题：无限大平面挖孔问题

**2.1 题目分析**
*   **题目解读**：这道题有点像绕口令。“该点场强大小的一半，是由平面上一个半径为 $R$ 的圆面积范围内的电荷产生的”。
*   **翻译成人话**：在距离无限大平面 $a$ 的地方，整个无限大平面产生的电场是 $E_{\text{总}}$。而在平面上画一个半径为 $R$ 的圆盘，这个圆盘单独产生的电场 $E_{\text{圆盘}}$，刚好等于 $E_{\text{总}}$ 的一半。求 $R$ 是多少。
*   **思路**：列方程，$E_{\text{圆盘}} = \frac{1}{2} E_{\text{总}}$。

**2.2 解题步骤**

1.  **写出无限大平面的场强公式**：
    $$E_{\text{总}} = \frac{\sigma}{2\varepsilon_0}$$

2.  **写出圆盘的场强公式**：
    $$E_{\text{圆盘}} = \frac{\sigma}{2\varepsilon_0} (1 - \frac{a}{\sqrt{a^2+R^2}})$$
    *(注：这里 $\frac{a}{\sqrt{a^2+R^2}}$ 就是 $\cos\theta$，其中 $\theta$ 是圆锥半顶角)*

3.  **根据题意列方程**：
    题目说 $E_{\text{圆盘}} = \frac{1}{2} E_{\text{总}}$，代入公式：
    $$\frac{\sigma}{2\varepsilon_0} (1 - \frac{a}{\sqrt{a^2+R^2}}) = \frac{1}{2} \cdot \frac{\sigma}{2\varepsilon_0}$$

4.  **化简求解**：
    两边同时消去 $\frac{\sigma}{2\varepsilon_0}$：
    $$1 - \frac{a}{\sqrt{a^2+R^2}} = \frac{1}{2}$$
    
    移项：
    $$\frac{a}{\sqrt{a^2+R^2}} = \frac{1}{2}$$
    
    两边平方：
    $$\frac{a^2}{a^2+R^2} = \frac{1}{4}$$
    
    交叉相乘：
    $$4a^2 = a^2 + R^2$$
    $$3a^2 = R^2$$
    $$R = \sqrt{3}a$$

    **答案是 $R = \sqrt{3}a$**，与参考答案一致。

### 第三部分：考前速记总结 (可直接复制到笔记)

这里为你整理了可以直接背诵和模仿的解题模板。

#### 1. 核心公式表
*   **库仑定律**：$F = k \frac{q_1 q_2}{r^2}$
*   **电场定义**：$E = \frac{F}{q_0} = k \frac{Q}{r^2}$
*   **无限大均匀带电平面场强**：$E = \frac{\sigma}{2\varepsilon_0}$
*   **均匀带电圆盘轴线场强**：$E = \frac{\sigma}{2\varepsilon_0} (1 - \cos\theta)$，其中 $\cos\theta = \frac{x}{\sqrt{x^2+R^2}}$

#### 2. 第4题：直杆与点电荷 (一维积分)
*   **模型**：长 $l$ 的带电杆，距离 $d$ 处的点电荷。
*   **关键步骤**：
    1. 设变量 $x$ 为微元到点电荷的距离。
    2. 微元电荷 $dq = \lambda dx$。
    3. 积分区间：$[d, d+l]$。
*   **计算式**：
    $$F = \int_{d}^{d+l} k \frac{q_0 \lambda}{x^2} dx = k q_0 \lambda [\frac{-1}{x}]_{d}^{d+l}$$
*   **最终结果**：
    $$F = k q_0 \lambda (\frac{1}{d} - \frac{1}{d+l})$$

#### 3. 第5题：半圆环圆心场强 (对称性积分)
*   **模型**：半径 $R$，带电量 $Q$ 或密度 $\lambda$ 的半圆。
*   **关键步骤**：
    1. 对称性分析：平行直径的分量抵消，只算垂直分量。
    2. 垂直分量微元：$dE_y = dE \sin\theta = \frac{k \lambda R d\theta}{R^2} \sin\theta$。
    3. 积分区间：$[0, \pi]$。
*   **最终结果**：
    $$E = \frac{2k\lambda}{R}$$
    (若已知总电荷 $Q$，则 $E = \frac{2kQ}{\pi R^2}$)

#### 4. 第6题：平面与圆盘 (比例关系)
*   **逻辑**：题目问“一半场强对应的半径”。
*   **方程**：
    $$1 - \frac{a}{\sqrt{a^2+R^2}} = \frac{1}{2}$$
*   **结论**：
    $$R = \sqrt{3}a$$
